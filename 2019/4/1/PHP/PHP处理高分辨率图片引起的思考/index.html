上图的意思就是说，我们能使用的内存最大是8M，但是处理这个图片还需要额外的41bytes，就会导致内存不足，这是一个很严重的错误。在对图片进行操作前需要将图片的所有信息读入内存中，同时还会使用另一部分内存同于处理计算并缓存输出，所以内存大小的使用还是和图片的大小有关。
